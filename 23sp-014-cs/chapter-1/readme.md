# Chapter 1 – Python Basics and Concepts

## Overview
This chapter covers fundamental Python concepts including classes, control flow, functions, file handling, data structures, and an introduction to parallel programming techniques.

---

## Topics Covered

### 1. classes.py
**Concept:** Classes and Objects

**Explanation:**
This file demonstrates how to create classes and objects in Python. It explains instance variables, class variables, and inheritance.

**Use:**
Used for modeling real-world entities and organizing code.

**Pros:**
- Code reuse
- Better structure
- Supports inheritance

**Cons:**
- Slightly complex for beginners

### 2. dir.py & flow.py
**Concept:** Control Flow (if-else, loops)

**Explanation:**
These files demonstrate conditional statements and loops such as `for` and `while`.

**Use:**
Used for decision-making and repeating tasks.

**Pros:**
- Core programming logic
- Easy to understand

**Cons:**
- Can become complex in large programs

Useful for reusable logic and modular code.
